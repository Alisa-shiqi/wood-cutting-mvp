# MVP Roadmap

## V0.1 Static Validation

Goal: prove the smallest useful workflow.

- Manual part input
- CSV import/export
- Automatic nesting
- Cutting layout
- Cut list
- Print

## V0.2 Trial Version

Goal: test with real orders from small workshops.

- Material/thickness grouped nesting: done
- Max-rect style free-space splitting: started
- Project JSON import/export: done
- Unplaced reason hints: done
- Edge banding fields: done
- Full-width part list workspace: done
- Ordered production workspace layout: done
- Print cleanup for workshop handoff: done
- Fixed-height production screen: done
- Stable two-column workspace shell: done
- Material/thickness color grouping: started
- Prominent group badges: started
- Three-column operation workspace: started
- Board group columns and material dropdowns: started
- Dense three-column tuning: started
- Edit-dirty result status and categorized presets: started
- Board category naming and dirty action: started
- Fullscreen nesting review: started
- Global part serials and offcut detail tables: started
- Result/offcut numbering cleanup: started
- Per-sheet code lists and final label sections: started
- Live part-list summary: started
- Reusable offcut hints: done
- Better print style: done
- Project file round-trip tests

## V0.3 Workshop-Usable Version

Goal: get first paid pilot.

- Order history: started
- Stock panel library: started
- Offcut library
- Label printing: started
- Print-to-PDF report export: started
- Native PDF export
- Basic DXF export: started
- Local trial guide: done
- Sample order library: started
- Pilot kit and demo script: done

## V1.0 Paid Version

Goal: turn it into a sellable small tool.

- Local license or login
- Customer/order management
- Multiple nesting plan comparison
- Configurable cut list template
- Private deployment package
- Trial/license mechanism

## Not In Early Scope

- Direct machine control
- Arbitrary image recognition
- Full CAD editor
- Multi-tenant SaaS
- Full custom-furniture design-to-manufacturing suite
