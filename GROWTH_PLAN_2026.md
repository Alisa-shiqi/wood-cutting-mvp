# Growth Plan 2026

> Local working draft. Do not commit or publish before founder review.

## 1. Goal

Land the product in two stages:

1. Traffic first: prove that woodworkers, board-cutting shops, and custom furniture buyers are willing to try the tool.
2. Revenue second: convert repeated users through paid setup, template configuration, and Pro features.

The product should not start as a broad SaaS. It should start as a practical tool with clear entry paths for different users.

## 2. User Segments

### TO B: Shops and Small Factories

Target users:

- Board-cutting shops
- Custom cabinet shops
- Small panel-furniture factories
- Decorator supply shops
- Local cabinet/wardrobe installers

Primary pain:

- Quote is slow or inaccurate.
- Cut list handoff is error-prone.
- Workers need printable instructions.
- Order status and delivery dates are poorly tracked.
- Existing CAD/CAM tools are too heavy.

Core promise:

Generate quote, part list, nesting layout, labels, and production order from one browser workflow.

### TO C: Homeowners, DIY Users, and Small Designers

Target users:

- People wanting a custom bookcase/desk/wardrobe
- DIY woodworking users
- Small interior designers
- Content viewers who want to estimate custom furniture cost

Primary pain:

- Do not know how many boards are needed.
- Do not know whether a quote is reasonable.
- Cannot translate a simple cabinet idea into a board list.

Core promise:

Choose a template, enter dimensions, estimate material and cost, then generate a cut list that can be taken to a local shop.

## 3. Traffic Strategy

### TO B Acquisition

Best channels:

- Offline visits to board-cutting shops and custom cabinet shops.
- WeChat groups for carpenters, cabinet makers, and renovation workers.
- Short demo videos sent directly to shop owners.
- Partnerships with board material sellers and CNC/edge-banding machine sellers.

Content formats:

- "3 minutes from customer size to cut list"
- "How to avoid quote loss on custom cabinets"
- "Print a production order workers can understand"
- "Same order: manual calculation vs software workflow"

First conversion action:

Ask for one real order and reproduce it in the tool for free.

Success metric:

The shop owner agrees the printed output can be used after small adjustments.

### TO C Acquisition

Best channels:

- Douyin/Bilibili/Xiaohongshu demo videos
- Search-friendly pages for "bookcase board calculator", "wardrobe material calculator", "custom desk quote estimator"
- Shareable design result pages in the future
- Downloadable cut-list/quote PDF in the future

Content formats:

- "Enter width/height/depth to calculate bookcase boards"
- "How much board does a 2.4m wardrobe need?"
- "DIY desk material estimate"
- "Take this cut list to a board shop"

First conversion action:

Use a free calculator/template and export a cut list.

Success metric:

User exports or prints a quote/cut list.

## 4. Revenue Strategy

### Stage 1: Service Revenue

Most realistic early revenue:

- Configure material prices for a shop.
- Build shop-specific templates.
- Customize print format.
- Train owner/workers.

Suggested pricing:

- RMB 499-999: basic setup
- RMB 999-1999: templates + price library + print format
- RMB 1999-3999: small shop workflow package

### Stage 2: Subscription

Suggested tiers:

- Free: limited templates, local history, basic nesting
- Personal: RMB 19-39/month, more history and exports
- Shop Pro: RMB 99-199/month, price library, branded quote, production packet
- Factory Lite: RMB 299-599/month, batch orders, offcut inventory, advanced exports

### Stage 3: Template Marketplace

Later revenue:

- Paid wardrobe/cabinet template packs
- Shop-specific template customization
- Industry-specific print forms

## 5. Product Packaging

### Quick Cutting Mode

For TO B users who already have dimensions.

Must stay simple:

- Default stock
- Part list
- Quote quick calc
- Nesting result
- Layout
- Cutting history

Do not show:

- Customer quote document
- Production work order
- Order status
- Complex dashboards

### Design Generation Mode

For TO C/new users and shop sales.

Must stay guided:

- Template selection
- Core dimensions
- Generated part list
- Quote estimate
- Nesting result
- Design history

Do not show:

- Order status
- Complex order dashboard
- Full production paperwork by default

### Professional Mode

For paid/power users.

Show full workflow:

- Order info
- History dashboard
- Quote document
- Production work order
- Exports
- Import/export
- Labels
