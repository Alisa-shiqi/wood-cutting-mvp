# Execution Tasks 2026

> Local working task plan. Do not commit or publish before founder review.

## Objective

Build a practical tool that first attracts real users, then converts shops through setup service and Pro features.

## Phase 1: Usable Trial Tool

Target time: 1-2 weeks

Goal:

Let users complete one real workflow without guidance.

Tasks:

1. Mobile usability
   - Full-page scrolling
   - Single-column forms
   - Working select controls
   - Local horizontal scroll for tables
   - Test on Chrome mobile emulator and real phone

2. Material price library
   - Preset board prices by material/thickness
   - Edge-banding price
   - Labor price
   - Waste/profit presets
   - One-click apply to quote

3. Quick cutting mode polish
   - Keep only default stock, quote, cutting history, part list, result, layout
   - Make cutting history independent from order history
   - Print-friendly cutting output

4. Design generation mode polish
   - Template-first flow
   - Hide complex order fields
   - Design history independent from order history
   - Better template preview

Validation:

- One cutting sample can be loaded, nested, quoted, and printed.
- One design sample can generate parts, nest, quote, and save design history.
- Mobile select and page scrolling work.

## Phase 2: Revenue-Critical Features

Target time: 2-4 weeks

Goal:

Make the tool credible enough for shop owners to pay for setup.

Tasks:

1. Wardrobe template
   - Width/height/depth
   - Columns
   - Shelves
   - Doors
   - Drawers
   - Back panel
   - Edge rules

2. Branded quote
   - Shop name
   - Phone
   - Validity period
   - Customer confirmation
   - Export/print

3. Production packet
   - Cutting checklist
   - Edge-banding checklist
   - Labels
   - Offcut warehouse table
   - Worker sign-off

4. Professional menu
   - Load sample
   - Save local
   - Import project
   - Export project
   - Keep hidden from beginner modes

Validation:

- Three realistic cases can be demonstrated: cutting shop, bookcase desk, wardrobe.
- A shop owner can understand quote and production packet without explanation.

## Phase 3: Pilot Acquisition

Target time: 4-8 weeks

Goal:

Get real orders from users and convert at least one paid setup.

TO B tasks:

1. Visit/contact 10 board-cutting or cabinet shops.
2. Reproduce one real order per shop.
3. Compare manual workflow vs tool workflow.
4. Collect printout feedback.
5. Offer setup package.

TO C tasks:

1. Publish demo clips:
   - Bookcase material estimate
   - Wardrobe board estimate
   - DIY desk cut list
2. Create shareable sample outputs.
3. Track exports/prints as conversion signal.

Validation:

- 10 real-order tests
- 3 usable case studies
- 1-3 paid setup conversations
